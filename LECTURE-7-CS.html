Rules for keywords:

1.     All keywords are lowercase words
2.     The same keywords cannot be used together in the same statement
3.     Keywords cannot be used as a name of variable, method, class, interface, namespace, etc.
i.e. Keywords  cannot be used as identifiers

Questions: Identifiers

1.     What is Identifier
2.     What is the need of Identifier
3.     Importance of Identifiers
4.     What are rules for using Identifiers in C#
5.     Can we write and compile a C# program without using any Identifier of C# programming language

Rules for Identifiers:

1.     Can be in upper case or lower case
2.     Can be of any length
3.     Can not begin with a  number
4.     Underscore(_) is allowed
5.     There should not be space between two word – which define identifier
6.     The name may consist of letters, digits, and underscore
7.     They must not begin with a digit
8.     It should not a keyword
9.     White space are not allowed
