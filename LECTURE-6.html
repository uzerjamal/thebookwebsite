HOW TO MAKE .DLL AND USE IT:

Step-1:write following code in code editor and save as "a.cs"

using System;

   public class A
    {
        public void display()
        {
            Console.WriteLine("Hello World!");
            Console.WriteLine("Press any key to exit.");
            Console.ReadKey();
        }
    }

Step-2:open DOS command prompt and type following command to make .dll

C:\>csc /target:library   a.cs

Step-3:write following code in code editor and save as "hello.cs"

using System;
    public class Hello
    {
        static void Main()
        {
            A x=new A();
              x.display();
            Console.WriteLine("Press any key to exit.");
            Console.ReadKey();
        }
    }

Step-4: use the following command to use the .dll in application created in step-3.

C:\>csc /reference:a.dll hello.cs
OR
C:\>csc /reference:a.dll /out:test.exe hello.cs
