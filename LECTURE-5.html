.NET FRAMEWORK FIRST PROGRAM
                                             
·         .NET framework comes along with windows7 operating system (true/false).
·         How to configure .NET framework for command line
·         Write, compile and execute first program from command line.


CONFIGURE .NET FRAMEWORK FOR COMMANDLINE

1.     open DOS command prompt
2.     cd\
3.     echo %path%
4.      set path = %path%;C:\Windows\Microsoft.NET\Framework\v3.5;

    Now you can compile and execute any program written in .net compatible language


FIRST PROGRAM IN C#.NET:

using System;
 class Hello
    {
        static void Main()
        {
            Console.WriteLine("Hello World!");
            Console.WriteLine("Press any key to exit.");
            Console.ReadKey();
        }
    }


1.     Make a folder in any of the drive ( C:/myprograms)
2.     Write the above program in notepad editor
3.     Save the file as  : hello.cs // it is a source code
4.     Goto DOS command prompt and configure the .net framework path
5.     C:\myprograms>csc hello.cs // this command will create  hello.exe file
6.     C:\myprograms>hello // or hello.exe


FIRST PROGRAM IN VB.NET:

Imports System
Module Module
   Sub Main()
      Console.WriteLine("Hello World")
      Console.ReadKey()
   End Sub
End Module


1.     Make a folder in any of the drive ( C:/myprograms)
2.     Write the above program in notepad editor
3.     Save the file as  : hello.vb // it is a source code
4.     Goto DOS command prompt and configure the .net framework path
5.     C:\myprograms>vbc hello.vb // this command will create  hello.exe file
6.     C:\myprograms>hello // or hello.exe
