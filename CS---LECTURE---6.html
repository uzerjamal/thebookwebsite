Assemblies: concepts

1.         Assemblies = MSIL + Metadata + manifest+ version information+ definition of types
2.        It provide assistance to CLR in code execution by providing information  needed at runtime
3.        There are two types of assemblies – private and public
4.        Private assembly is related to only one application, it cannot be share among multiple application or projects
5.        Public assembly can be shared among multiple application or project
6.        Private assembly is store in project or application hierarchy
7.        Public assembly stored in global assembly cache(GAC)  so that it can be shared among multiple application
8.        Only strong named assembly can be stored in GAC
