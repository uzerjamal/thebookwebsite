STRONGLY TYPED:

Any programming language where type is mandatory to define a variable, those languages are called strongly typed :

examples of storngly typed languages

1. Java
2. C#
examples of non-strongly typed languages:  javascript

 TYPE-SAFTEY:  

Type safety means that the compiler will validate types while compiling, and show an error if we try to assign the wrong type to a variable.

String s="India"; //correct, No Error
String s=123;     //wrong , ERROR

javascript is not type safe programming language: var a=5;var b="abc"

var s="India";// correct, NO ERROR
var s=123;    // correct, NO ERROR


C# VERSIONS AND FEATURES:

C# version 1.0:
                        - Classes
                        - Structs
                        - Interfaces
                        - Events
                        - Properties
                        - Delegates
                        - Expressions
                        - Statements
                        - Attributes

C# version 1.2 :Visual Studio 2003.

                - foreach loop
                - IEnumerator


C# version 2.0: released in 2005, along with Visual Studio 2005:

                        - Generics
                        - Partial types
                        - Anonymous methods
                        - Nullable types
                        - Iterators
                        - Covariance and contravariance
                        - Getter/setter separate accessibility
                        - Method group conversions (delegates)
                        - static classes
                        - Delegate inference



C# version 3.0 : came in late 2007, along with Visual Studio 2008,
                          come with .NET Framework version 3.5.


                                - Auto-implemented properties
                    - Anonymous types
                                - Query expressions
                                - Lambda expressions
                                - Expression trees
                               - Extension methods
                               - Implicitly typed local variables
                               - Partial methods
                               - Object and collection initializers

C# version 4.0:

                        - Dynamic binding
                        - Named/optional arguments
                        - Generic covariant and contravariant
                        - Embedded interop types

C# version 5.0:

                        - Asynchronous members
                        - Caller info attributes


C# version 6.0:

                        - Static imports
                        - Exception filters
                        - Auto-property initializers
                        - Expression bodied members
                        - Null propagator
                        - String interpolation
                        - nameof operator
                        - Index initializers

C# version 7.0:

                        - Out variables
                        - Tuples and deconstruction
                        - Pattern matching
                        - Local functions
                        - Expanded expression bodied members
                        - Ref locals and returns
                        - Other features included:
                        - Discards
                        - Binary Literals and Digit Separators
                        - Throw expressions

C# version 7.1: Visual Studio 2017 version 15.3 and .NET Core 2.0 SDK.

C# version 7.2: Visual Studio 2017 version 15.5, and .NET Core 2.0 SDK.

C# version 7.3: Visual Studio 2017 version 15.7 and .NET Core 2.1 SDK.
