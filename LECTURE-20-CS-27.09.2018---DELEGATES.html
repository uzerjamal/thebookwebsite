PROGRAMMING CONCEPTS
 DELEGATES IN C#


•         Delegates (Points):
•         Delegates are reference to methods.
•         Delegates are typesafe function pointers.
•         Delegates points to methods of perticular type.
•         Delegates can be created with parameter or without parameter
•         signature of delegate and methods must be same.
•         Delegate for static method
•         Delegate for non static method
•         Delegate with return type of method
•         Delegate with void type of method
•         How to make delegates
•         Single cast delegate
•         Multicast delegates
•         Multicast delegate are made for methods – return type is void.
•         All methods for multicast delegate must be same syntax.
•         delegates can be declared with in the class or outside the class
•         delegates stores referece of methods ( can be static or non static)
•         delegates are reference type
•         what is the need or importance of delegates : event driven approach
•         one delegate can store refereces of more than one methods : called multicast delegate
•         System.Delegate class

Program-4: calling a static method using delegate : syntax -1

using System;
public delegate void mydel();
class Program
{
    static void Main()
    {
        mydel md=new mydel(A.display);// calling static method, SYNTAX - 1
        md();
        Console.ReadKey();
    }
}
